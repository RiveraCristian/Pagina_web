# Imágenes de Proyectos

Esta carpeta contiene las imágenes de los proyectos.

Para agregar imágenes reales de tus proyectos:

1. Coloca las imágenes en esta carpeta: `public/img/proyectos/`
2. Los nombres deben coincidir con los especificados en `src/data/projects.ts`:
   - pri1.png
   - pri2.png
   - omed1.png
   - edu1.png
   - analytics1.png

Las imágenes recomendadas deben tener:
- Formato: PNG o JPG
- Tamaño recomendado: 800x600px o similar (ratio 4:3)
- Peso: menos de 500KB por imagen

Si las imágenes no existen, la aplicación mostrará un placeholder automático.
