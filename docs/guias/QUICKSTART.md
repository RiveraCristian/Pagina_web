# Quick Start - Inicio Rápido

## ⚡ Cómo probar la aplicación en 3 pasos

### Paso 1: La aplicación ya está corriendo
El servidor de desarrollo está activo en: **http://localhost:5174/**

Si no está corriendo, ejecuta:
```bash
npm run dev
```

### Paso 2: Abre tu navegador
1. Ve a: http://localhost:5174/
2. Verás el logo "COLABI" centrado en fondo negro

### Paso 3: Interactúa
1. **Haz clic** en el logo
2. **Escribe** una de estas frases de ejemplo:
   - "soy corredor de seguros"
   - "trabajo en un colegio"
   - "necesito un dashboard"
   - "trabajo en el gobierno"
3. **Presiona Enter**
4. ¡Mira cómo aparecen los proyectos relevantes!

---

## 🎨 Personalización Rápida

### Cambiar el nombre de la empresa
Edita: `src/components/LogoHero.tsx` línea 10

### Agregar tus proyectos
Edita: `src/data/projects.ts`

### Agregar imágenes
Coloca tus imágenes en: `public/img/proyectos/`
