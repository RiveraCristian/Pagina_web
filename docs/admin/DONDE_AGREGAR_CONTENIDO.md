# 📝 GUÍA VISUAL: Dónde Agregar Contenido en el Admin Panel

## 🎯 **PASO A PASO PARA AGREGAR MISIÓN, VISIÓN Y ENLACES**

### **PASO 1: Acceder al Admin Panel**
```
🌐 Ir a: http://localhost:5173/admin
🔑 Contraseña: admin123
📋 Click: "Contenido" (en el menú lateral izquierdo)
```

### **PASO 2: Seleccionar la Categoría "Nosotros"**
```
📁 En el dropdown "Seleccionar categoría": 
   ▼ Seleccionar: "Sobre Nosotros"
```

### **PASO 3: Crear Nuevo Contenido**
```
➕ Click: "+ Nuevo Elemento"
```

## 🔄 **FLUJO COMPLETO DE CREACIÓN**

### **Paso 1: Crear Misión**
```
Admin > Contenido > "Sobre Nosotros" > + Nuevo Elemento
→ Llenar formulario de Misión
→ Guardar
```

### **Paso 2: Crear Visión**
```
Admin > Contenido > "Sobre Nosotros" > + Nuevo Elemento  
→ Llenar formulario de Visión
→ Guardar
```

### **Paso 3: Crear Quiénes Somos**
```
Admin > Contenido > "Sobre Nosotros" > + Nuevo Elemento
→ Llenar formulario de Quiénes Somos  
→ Guardar
```

### **Paso 4: Verificar en el Frontend**
```
http://localhost:5173/
🔍 Buscar: "misión" → Debe aparecer tu contenido
🔍 Buscar: "visión" → Debe aparecer tu contenido
🔍 Buscar: "colabi" → Debe mostrar todo
```
